# Titanic Survival Analysis Dashboard

## Project Overview
The sinking of the RMS Titanic in 1912 remains one of the most well-known maritime disasters in history. This project analyzes passenger demographics, ticket information, and travel patterns to identify the major factors that influenced survival rates.

The analysis focuses on:
- Gender
- Passenger Class
- Fare Category
- Family Size
- Embarkation Port
- Age Groups

---

## Data Cleaning
- Removed Cabin column due to high missing values
- Filled missing Age values using median
- Filled missing Embarked values using mode
- Checked duplicate records
- Prepared dataset for analysis

---

## Feature Engineering
- Created Family Size feature
- Created Fare Categories (General, Premium, Luxury)
- Created Age Groups (Children, Adults, Seniors)

---

## Key Insights

### Survival Rate by Gender
Female passengers had significantly higher survival rates than male passengers.

### Survival Rate by Passenger Class
First-class passengers had better survival chances compared to lower classes.

### Passenger Class + Gender Analysis
First-class females had the highest survival rate, while third-class males had the lowest.

### Fare Analysis
Passengers paying higher fares showed better survival probabilities.

### Family Size Analysis
Passengers traveling in smaller families had better survival outcomes.

### Embarkation Port Analysis
Survival rates varied across different boarding ports.
